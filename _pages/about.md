I am a data and applied scientist at Microsoft, where I analyze M365 Copilot usage patterns to drive adoption. I received my Ph.D. from [Rochester Institute of Technology](https://www.rit.edu/).  

My dissertation research -- [Towards Algorithm Selection for Efficient Search-Based Software Engineering](https://repository.rit.edu/theses/11786/) -- focused on algorithm selection for software engineering tasks. The goal of my research was recommend the best-fit optimization algorithms for specific problem instances, enabling practitioners to build and maintain software systems more effectively. My work combined empirical analysis of evolutionary algorithm behavior on hard SE problems, developing predictive performance models, and using those models to automate algorithm selection for more effective and efficient software engineering.

I've applied these techniques to service-oriented systems and automated API recommendations during library migration. Our paper on [self-adaptive service-oriented systems using algorithm selection](https://ieeexplore.ieee.org/document/9590318) won the [best paper award at IEEE ICWS 2021](https://conferences.computer.org/services/2021/awards/).​
